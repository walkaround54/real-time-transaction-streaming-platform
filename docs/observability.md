# Observability

## Goals

Monitor both platform health and business performance.

## Prometheus

Collect metrics from:

- Kafka
- Flink
- Kubernetes
- Producer
- JVM

## Grafana Dashboards

Infrastructure:

- CPU
- Memory
- Consumer lag
- Throughput
- Latency
- Checkpoint duration
- Failed checkpoints

Business:

- Transactions/min
- Total amount
- Failed transactions
- Fraud alerts
- Alerts by severity
- Alerts by rule
